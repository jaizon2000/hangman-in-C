# hangman-in-C
Hangman game in C

## Tools Used:
+ Emacs
+ Oracle VM Virtual Box

This program must be run on a Linux system because it takes words from `/usr/share/dict/words/`.  
I used an Ubuntu Oracle VM Virtual Box to create this project.
